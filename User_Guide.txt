For DPT 2.3

Manual for creating drawings and images.

1. Make your screen and import the files
  
  Choose a variable to assign your window to, where it is (defaults to 0,0), its dimentions and what the display name will be (defaults to 'DXL')
mkcanvas(variablename,length,width,xy=(x,y),screenname='DXL')

from dpged import *
from screenkitlayers import *
mkcanvas(MyCanvas,300,300,xy=(300,300),screenname='My First Canvas')

2. Make some layers!

 Layer(priority (higher = higher priority (move to front), xytocolour (what's initially on layer, [(item: coordinates(x,y), colour(r,g,b) in decimal)]), defaults to nothing)
 
 background(-1)
 bottomlayer(0)
 middlelayer(1)
 toplayer(2)
 bottommiddlelayer(0.5)
 
 3. Draw!
 
 Draw on layers: 
  sizeadd(...)      draws    squares  - see ScreenKitLayers documentation    <---
  rectadd(...)      draws rectangles  -              ||
  multiadd(...)     draws multiple px -              ||                      <---
  [...].setpx(...)  draws one pixel   -              ||
  
 Stencil layers onto canvas:
  update(...) - see ScreenKitLayers documentation          <---
  
 Markup directly on canvas:
  fill(...) - see ScreenKitLayers documentation            <---
  sizeprint(...) -          ||                             <---
  rectprint(...) -          ||
  [...].drawpx(...)  - see DPGED documentation
  [...].drawsq(...)  -          ||
  [...].drawrt(...)  -          ||
